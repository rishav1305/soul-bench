# soul-eval

## Overview
Statistical evaluation framework — bootstrap confidence intervals, paired t-tests, report generation.

## Status
**To Implement** | Month 3-4 (Phase 7: Research Portfolio)

## Description
soul-eval provides rigorous statistical comparison of model performance. It takes benchmark results from soul-bench and produces publication-quality statistical analysis with proper significance testing.

## Key Features
- Bootstrap confidence intervals
- Paired t-tests for model comparison
- Automated report generation
- Statistical significance testing
- Visualization of results with error bars
- LaTeX table generation for papers

## Relationship to Other Projects
- **soul-bench**: Provides raw benchmark results that soul-eval analyzes
- **soul-moe**: Model selection decisions informed by soul-eval analysis

## Output
- Publishable statistical comparisons
- Automated report generation
- Reproducible analysis pipelines

## Portfolio Signal
Research credibility — demonstrates proper statistical methodology for AI evaluation.
