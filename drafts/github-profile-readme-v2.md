**Engineer** — Built an enterprise agentic AI platform at Gartner serving 5,000+ concurrent users. Engineered data platforms at Novartis across 15 pharmaceutical brands.

### Claude Code Power User

My entire AI operating system uses Claude as its brain. I maintain **14 Claude Code subagents** (5 for development, 9 for operations), enforce architecture through **hookify rules**, and route all LLM calls through a centralized client supporting Claude API, CLI, and multi-model fallback. Every line of soul-os was written with Claude Code.
